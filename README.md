# simplecooker_back

#### Simple cook book application with react
## Features
* Fully adaptive design
* User friendly ux

## Technologies
Back-end:
* Spring framework
* Assembly by Maven

Front-end:
* React.js, React Router
* Bootstrap
* Assembly by React scripts

## System requirements
* Java 8
* Hibernate compatible relational database (_PostgreSQL_ used)
* Maven for building

## Getting started
Back-end:
1. `git clone https://github.com/FoxCrazy/simplecooker_back.git`
2. `mvn clean install`
3. launch

Front-end:
1. `mvn install`
2. `npm start`
